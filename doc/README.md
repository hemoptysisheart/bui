# BUI(Block UI)

## 디렉토리

- `db` : DB 관련 파일. ERD, `*.sql` 등.
- `doc` : 프로젝트 문서.
- `web-front` : HTML 템플릿 등 웹 프론트 리소스용 디렉토리.

## I18N

### 메시지

- `components.` : 여러 페이지에서 공유하는 UI 컴포넌트의 메시지.
- `page.` : 특정 URL의 페이지에서만 사용하는 메시지. 페이지 타이틀 등.

### 에러 메시지

- `err.page.` : 특정 페이지에 한정된 에러 메시지.

## DB

### 테이블 이름

- `op_` : 관리자 계정에 관한 정보.
- `ui_` : UI 구성에 필요한 정보.
